## Properties

Though this is made for APIs, there may be some tags or properties you'd like to style similarly. Here, use the `<Properties />` warpper with `<Property />`  fields:

### Examples of Properties

<Properties>
  <Property name="id" type="string">
    Unique identifier for the contact.
  </Property>
  <Property name="last_active_at" type="timestamp">
    Timestamp of when the contact was last active on the platform.
  </Property>
</Properties>

## Code highlighting:

Use a `<CodeGroup />` component and include a label between `{{ }}`. Use `title` to label something freely, use `language` for programming languages.

<CodeGroup>

```bash {{ title: 'cURL' }}
# cURL is most likely already installed on your machine
curl --version
```

```js {{ language: 'js' }}
console.log('hello world')
```
</CodeGroup>
